# Product Overview

Yii 2 Basic Project Template - a skeleton web application for rapidly creating small PHP projects.

## Core Features
- User authentication (login/logout)
- Contact form with email support
- Basic page structure (home, about, contact, error pages)

## Target Use Case
Starting point for small to medium PHP web applications using the Yii2 framework.
